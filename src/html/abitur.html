Абитуриентам

Приемная комиссия филиала МАИ проводит прием документов:

с 20 июня по 25 июля для поступащих по результатам ЕГЭ, с 20 июня по 10 июля для лиц, участвующих во внутренних вступительных испытаниях.

МАИ осуществляет прием граждан сверх контрольных цифр приема для обучения на основе договоров с оплатой стоимости обучения.

Осуществляется целевой набор для предприятий оборонно-промышленного комплекса и предприятий региона.

Для юношей, годных к строевой службе, возможно обучение на военной кафедре.

Подготовительные курсы

Ступинский филиал МАИ объявляет набор на подготовительные курсы для подготовки к сдаче экзаменов в форме ЕГЭ.

Подготовительные курсы проводятся для школьников 11-х классов и выпускников техникумов по различным предметам для успешной сдачи ЕГЭ и участия в олимпиадах.

Изучаемые предметы:

  • математика
  • физика
  • русский язык

Как поступить в Ступинский филиал МАИ

Шаг 1 - Сдать ЕГЭ

Для поступления на бакалавриат сдайте ЕГЭ по общеобразовательным предметам.

Перечень вступительных испытаний и минимальные баллы можно посмотреть на странице Приемная компания

Шаг 2 - Подать документы

  • лично или доверенным лицом (по адресу: Московская область, г. Ступино, ул. Пристанционная, д. 4., ауд.203)
  • через личный кабинет абитуриента
  • через портал Госуслуг
  • через операторов почтовой связи общего пользования (на адрес: 142800, Московская область, г. Ступино, ул. Пристанционная, д. 4)

Шаг 3 - Сдать экзамены и пройти по конкурсу

Вступительные испытания проводятся для лиц, не имеющих результатов ЕГЭ со средним профессиональным или высшим образованием, инвалидностью, иностранным гражданством, проходивших итоговую аттестацию в иностранных образовательных учреждениях.

Зачисление проводится по конкурсным (ранжированным) спискам. Для зачисления поступающий подает заявление о согласии на зачисление.

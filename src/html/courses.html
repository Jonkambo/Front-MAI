Подготовительные курсы

Ступинский филиал МАИ объявляет набор на подготовительные курсы для подготовки к сдаче экзаменов в форме ЕГЭ.

Подготовительные курсы проводятся для школьников 11-х классов и выпускников техникумов по различным предметам для успешной сдачи ЕГЭ и участия в олимпиадах.

Занятия проводят опытные преподаватели - эксперты по проведению экзаменов в форме ЕГЭ.

Расписание занятий

Занятия проводятся 1 раз в неделю по 3 ак.часа по каждому предмету с 16:30.

Изучаемые предметы:

  • математика;
  • физика;
  • русский язык.

Стоимость

Стоимость одного предмета 3600 руб. в месяц (12 академических часов).

Приём документов

на подготовительные курсы проводится с 4 сентября по 30 сентября с 14:00 до 16:00 по будним дням.

Для заключения договора необходимо присутствие законного представителя (родителя) с паспортом.

При себе иметь 2 фотографии 3х4 (или 4х6) и паспорт.

Контакты

Телефон приёмной комиссии: +7 (496) 641-16-30
